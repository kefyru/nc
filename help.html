1. Установить приложение TestFlight
  2. Скопировать эту ссылку:
    https://raw.githubusercontent.com/kefyru/nc/refs/heads/main/appinfo.xml
  3. Установить приложение UIX для тестирования
  4. Запустить приложение UIX, в верхнем левом углу нажать плюсик
    Вставить в поле скопированную ссылку, нажать ввод. Через несколько секунд появится приложение NC, жмем Install, запускаем
  5. В поле "логин" вводим номер телефона в формате +71231231212
    Звоним на хелп что бы получить код.
